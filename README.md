# fnyactional
A functional programming language

## Installation
This interpreter requires the [catcare](https://github.com/labricecat/catcaretaker) program to be installed!
```
$ git clone https://github.com/labricecat/fnyactional
$ cd fnyactional
$ catcare sync
$ mkdir build
$ cmake .. && make install
$ ./fnya help
```
